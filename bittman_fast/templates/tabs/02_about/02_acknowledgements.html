ahx-recipe-randomizer


A demonstration project to show that web development without frameworks and Javascript is possible.
The only tools used are Python, Flask, Html, and CSS.

The tabbed interface is inspired by Jason Knight:Tabbed Interfaces Without JavaScript.

The recipes are from Mark Bittman's excellent book How to Cook Everything fast.

You can find the code on github. {% endblock content %}
